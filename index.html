Help & Documentation
FAQ & User Guide
Frequently Asked Questions
How do I add a new investment?
Navigate to Add Investment from the sidebar. Fill in the investment name, amount, buy price, sell price, and quantity. Click Add Investment to save it to your portfolio.
How is Profit/Loss calculated?
P&L = (Sell Price − Buy Price) × Quantity. For example, if you bought 30 shares at ₹2,400 and sold at ₹2,800: P&L = (2800 − 2400) × 30 = ₹12,000 profit. Use the P&L Calculator page for instant results.
How do I save my portfolio?
Go to Import / Export → click Save Now. Your data is stored in portfolio.dat in the application directory. You can also enable Auto-save in Settings.
Can I import data from Excel?
You can import CSV files, which can be exported from Excel. Go to Import / Export → Import Portfolio, and select your .csv file. Ensure it follows the format: Name, Amount, BuyPrice, SellPrice, Quantity.
How do I delete an investment?
In the Portfolio table, click the red trash icon on the investment row. A confirmation dialog will appear. Click Confirm to permanently delete the record.
Is there a backup feature?
Yes! Go to Settings → Data → enable "Backup on save". Each time you save, a .bak file is created alongside your portfolio.dat for recovery.
